# MP-SL3CI
Considering the personal privacy issues involved in users' forwarding text content, we release the users' commenting time data, which is collected from the Chinese Sina Weibo and Douyin using the Application Program Interface (API).
The accurate commenting time of users in these cases are shown in the file, which are the raw data without processing.

The model directory contains the code used in this study, including the personality classification code and the numerical fitting code. The personality classification code is a BERT-based toolkit for processing social media comments. It extracts key features to classify commenters’ personalities and mine their personality traits, supporting batch processing and outputting results for scenarios like user profiling and public opinion analysis. The numerical fitting code is used to generate the fitting curves reported in the manuscript and supports the reproduction of the main fitting results of the proposed MP-SL3CI framework.

The released code and data are provided for academic research, reproducibility verification, and non-commercial use only.
